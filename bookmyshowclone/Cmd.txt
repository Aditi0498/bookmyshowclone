Commands:
File Explorer
  >> git init

Vs Code Terminal:
  >> npm init
  >> npm i create-react-app
  >> npx create-react-app bookmyshowclone

  >> cd bookmyshowclone
  >> npm start

Axios:
  >> npm i axios

Headless UI:
  >> npm install @headlessui/react


React Icons:
